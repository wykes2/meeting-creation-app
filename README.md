# Meeting Creation App

A modern, responsive meeting creation app with 5-screen flow.

## Features
- Meeting type selection (1-on-1, Group, Team, etc.)
- Meeting details form with validation
- Participant selection with search
- Meeting summary and confirmation
- Success screen with meeting link

## Tech Stack
- React 19
- TypeScript
- Lucide React icons
- Custom CSS with modern design
- Deployed on Vercel
